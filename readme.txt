Projeto - Simulador Loja Digital:
    - Motivação:
        - Usar conceitos da orientação a objetos
        - Construir um sistema em Python que seja capaz de:
            - Para o cliente:
                - Adicionar items em um carrinho
                - Verificar items repetidos
                - Simular o sistema de pagamento de uma loja
                - Sistema de buscas de produtos
            - Para o gerente:
                - Cadastrar produtos
                - Verificar a entrada de capital
                - Sistema de buscas de produtos
                - Simular o recebimento do capital
